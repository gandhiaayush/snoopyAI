# Local Development Notes

These notes summarize setup signals discovered from repository metadata.

## Detected Tooling

- Node.js package metadata (`package.json`)

## Common Commands

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run check`
- `npm run start`

## Safety Notes

- Keep secrets in local environment files, not in tracked files.
- Prefer dry-run or read-only modes when testing automation.
